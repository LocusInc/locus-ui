## Examples

### Label Positions

The label can be positioned with the `position` prop: `top`, `left`, or `inside`.

<PreviewBox title="label-positions.tsx" code={labelPositionsSnippet}>
  {labelPositions.map((position) => (
    <Select.Root key={position} variant="outlined">
      <Select.Label position={position}>
        {position.charAt(0).toUpperCase() + position.slice(1)}
      </Select.Label>
      <Select.Trigger />
      <Select.Content>
        <Select.Item value="option1">Option 1</Select.Item>
        <Select.Item value="option2">Option 2</Select.Item>
        <Select.Item value="option3">Option 3</Select.Item>
      </Select.Content>
    </Select.Root>
  ))}
</PreviewBox>

### Variants

Select components come with three built in variants, `outlined`, `solid`, and `clear`, each shown across all label positions.

<PreviewBox title="variants.tsx" code={variantsSnippet}>
  {variants.map((variant) => (
    <Select.Root key={variant} variant={variant}>
      <Select.Label position="left">
        {variant.charAt(0).toUpperCase() + variant.slice(1)}
      </Select.Label>
      <Select.Trigger />
      <Select.Content>
        <Select.Item value="option1">Option 1</Select.Item>
        <Select.Item value="option2">Option 2</Select.Item>
        <Select.Item value="option3">Option 3</Select.Item>
      </Select.Content>
    </Select.Root>
  ))}
</PreviewBox>

### Groups

Items can be organized into groups using `Select.Group` with a `title`, separated by `Select.Separator`.

<PreviewBox title="groups.tsx" code={groupsSnippet}>
  <Select.Root variant="solid">
    <Select.Trigger />
    <Select.Content>
      <Select.Group title="Group 1">
        <Select.Item value="option1">Option 1</Select.Item>
        <Select.Item value="option2">Option 2</Select.Item>
      </Select.Group>
      <Select.Separator />
      <Select.Group title="Group 2">
        <Select.Item value="option3" disabled>
          Option 3
        </Select.Item>
        <Select.Item value="option4">Option 4</Select.Item>
      </Select.Group>
    </Select.Content>
  </Select.Root>
</PreviewBox>
