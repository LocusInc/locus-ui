## Examples

### Variants

Skeleton components come with three built-in variants: `solid`, `pulse`, and `shimmer`.

<PreviewBox title="variants.tsx" column code={variantsSnippet}>
  {variants.map((variant) => (
    <Grid key={variant} columns="2">
      <Text>{variant}</Text>
      <Skeleton variant={variant} style={{ width: 200, height: 24 }} />
    </Grid>
  ))}
</PreviewBox>

### Loading State

Use the `loading` prop to toggle between the skeleton and its children. When `loading` is `false`, the children are rendered instead of the skeleton placeholder.

<PreviewBox title="loading.tsx" code={loadingSnippet}>
  <Grid columns="2" gap="2">
      <Text>Loading</Text>
      <Skeleton
        loading={true}
        variant="pulse"
      >
        <Text>This is the content</Text>
      </Skeleton>

      <Text>Loaded</Text>
      <Skeleton
        loading={false}
        variant="pulse"
      >
        <Text>This is the content</Text>
      </Skeleton>
    </Grid>

</PreviewBox>

### Composition

Skeletons can be composed to represent more complex loading layouts.

<PreviewBox title="composition.tsx" code={compositionSnippet}>
  <Flex direction="column" gap="2">
    <Skeleton variant="pulse" style={{ width: 120, height: 12 }} />
    <Skeleton variant="pulse" style={{ width: 200, height: 12 }} />
    <Skeleton variant="pulse" style={{ width: 160, height: 12 }} />
  </Flex>
</PreviewBox>
