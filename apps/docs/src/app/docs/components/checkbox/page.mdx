## Examples

### Variants

Checkbox components come with three built in variants, `solid`, `outlined`, and `muted`.

<PreviewBox title="variants.tsx" code={variantsSnippet}>
  {variants.map((variant) => (
    <Checkbox.Root key={variant} variant={variant}>
      <Checkbox.Label>{variant}</Checkbox.Label>
    </Checkbox.Root>
  ))}
</PreviewBox>

### Label Positions

The label element `<label>` is available in the checkbox component anatomy. If included it can be easily positioned with the `position` prop.

<PreviewBox title="label-positions.tsx" code={labelPositionsSnippet}>
  {labelPositions.map((position) => (
    <Checkbox.Root key={position} variant="muted">
      <Checkbox.Label position={position}>
        {position.charAt(0).toUpperCase() + position.slice(1)} Label
      </Checkbox.Label>
    </Checkbox.Root>
  ))}
</PreviewBox>

### Sizes

To manage the size of the checkbox component you can provide a default or custom size through the `size` prop.

<PreviewBox title="sizes.tsx" code={sizesSnippet}>
  {sizes.map((size) => (
    <Checkbox.Root key={size} size={size} variant="muted">
      <Checkbox.Label>{size}</Checkbox.Label>
    </Checkbox.Root>
  ))}
  <Checkbox.Root size="40px" variant="muted">
    <Checkbox.Label>Custom Size</Checkbox.Label>
  </Checkbox.Root>
</PreviewBox>

### Colors

The checkbox component uses the `color` prop to allow for predefined and custom colors.

<PreviewBox title="colors.tsx" code={colorsSnippet}>
  {colors.map((color) => (
    <Checkbox.Root defaultChecked key={color} variant="muted" color={color}>
      <Checkbox.Label>
        {color.charAt(0).toUpperCase() + color.slice(1)}
      </Checkbox.Label>
    </Checkbox.Root>
  ))}
</PreviewBox>

### Alignments

To set alignment of the checkbox and its label content you can use the `align` prop.

<PreviewBox title="alignments.tsx" code={alignmentsSnippet}>
  {alignments.map((alignment) => (
    <Checkbox.Root
      key={alignment}
      align={alignment}
      variant="muted"
      className="w-1/4"
    >
      <Checkbox.Label>
        This checkbox is {alignment} aligned relative to its label.
      </Checkbox.Label>
    </Checkbox.Root>
  ))}
</PreviewBox>

### States

Checkboxes can be given the `disabled`, `readonly`, and `defaultChecked` states.

<PreviewBox title="states.tsx" code={statesSnippet}>
  <Checkbox.Root disabled variant="muted">
    <Checkbox.Label>Disabled</Checkbox.Label>
  </Checkbox.Root>
  <Checkbox.Root readonly variant="muted">
    <Checkbox.Label>Readonly</Checkbox.Label>
  </Checkbox.Root>
  <Checkbox.Root defaultChecked variant="muted">
    <Checkbox.Label>Default Checked</Checkbox.Label>
  </Checkbox.Root>
</PreviewBox>

### Indeterminate

Checkboxes support the `indeterminate` state for representing a partially selected state.

<PreviewBox title="indeterminate.tsx" code={indeterminateSnippet}>
  {variants.map((variant) => (
    <Checkbox.Root key={variant} variant={variant} indeterminate>
      <Checkbox.Label>{variant}</Checkbox.Label>
    </Checkbox.Root>
  ))}
</PreviewBox>

### High Contrast

Only available on the `"solid"` variant.

<PreviewBox title="high-contrast.tsx" code={highContrastSnippet}>
  <Checkbox.Root defaultChecked highContrast variant="solid">
    <Checkbox.Label>Solid</Checkbox.Label>
  </Checkbox.Root>
  <Checkbox.Root indeterminate highContrast variant="solid">
    <Checkbox.Indicator />
    <Checkbox.Label>Solid Indeterminate</Checkbox.Label>
  </Checkbox.Root>
</PreviewBox>
