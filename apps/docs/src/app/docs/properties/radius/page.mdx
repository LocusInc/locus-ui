# Radius Property

Controls the border-radius of elements. Supports scaled values, custom values, responsive breakpoints, inheritance, and individual corner control.

## Examples

### All Radii / Roundness

The `radius` prop accepts values `none`, `xs`, `sm`, `md`, `lg`, `xl`, and `full`. Combined with the `roundness` multiplier (`1`â€“`6`), this creates a full matrix of border-radius options.

<PreviewBox title="all-radii.tsx" code={allRadiiSnippet}>
  <Flex direction="column" gap="2">
    {numbers.map((number) => (
      <Flex key={number} gap="2">
        {radii.map((radius) => (
          <Box
            p="2"
            key={radius}
            radius={radius}
            roundness={number}
            className="border border-[rgb(var(--border-1))]"
          >
            <Text>
              {radius}, {number}
            </Text>
          </Box>
        ))}
      </Flex>
    ))}
  </Flex>
</PreviewBox>

### Inheritance

Radius values can be inherited from parent elements using `radius="inherit"`, or fall back to the theme default.

<PreviewBox title="inheritance.tsx" code={inheritanceSnippet}>
  <Box radius="none" p="2" className="border border-[rgb(var(--border-1))]">
    <Text>No Radius</Text>
    <Box radius="xl" p="2" className="border border-[rgb(var(--border-1))]">
      <Text>XL Radius</Text>
      <Box p="2" className="border border-[rgb(var(--border-1))]">
        <Text>Default/Theme</Text>
      </Box>
    </Box>
    <Box
      radius="inherit"
      p="2"
      className="border mt-2 border-[rgb(var(--border-1))]"
    >
      <Text>Inherit</Text>
      <Box p="2" className="border border-[rgb(var(--border-1))]">
        <Text>Default/Theme</Text>
      </Box>
    </Box>
  </Box>
  <Box
    radius="9px"
    p="2"
    mt="4"
    className="border border-[rgb(var(--border-1))]"
  >
    <Text>This box has a custom radius (9px)</Text>
    <Box
      radius="inherit"
      p="2"
      mt="2"
      className="border border-[rgb(var(--border-1))]"
    >
      <Text>Inherit</Text>
    </Box>
    <Box p="2" mt="2" className="border border-[rgb(var(--border-1))]">
      <Text>Default/Theme</Text>
    </Box>
  </Box>
</PreviewBox>

### Responsive

Radius supports responsive values using breakpoint objects.

<PreviewBox title="responsive.tsx" code={responsiveSnippet}>
  <Box
    p="2"
    my="4"
    radius={{ initial: "xs", sm: "xl", md: "full" }}
    className="border border-[rgb(var(--border-1))]"
  >
    <Text>
      This box has breakpoint-specific radii (xs initial, xl on sm, full on md)
    </Text>
  </Box>
</PreviewBox>

### Side Radii

Use `radius-t`, `radius-r`, `radius-b`, and `radius-l` to set radius on specific sides.

<PreviewBox title="side-radii.tsx" code={cornerRadiiSnippet}>
