# Padding Property

Controls padding for elements. Supports scaled values, custom values, and directional control.

  codeSnippets={[

## Examples

### Padding Scale

The `p` prop accepts scale values `0`–`8` for consistent spacing.

<PreviewBox title="padding-scale.tsx" code={paddingScaleSnippet}>
  <Flex gap="2" align="center">
    {paddingValues.map((num) => (
      <Box
        key={num}
        className="border border-[rgb(var(--border-1))] h-fit"
        p={`${num}`}
      >
        {num}
      </Box>
    ))}
  </Flex>
</PreviewBox>

### Padding Directions

Use `p`, `pt`, `pb`, `pl`, `pr`, `px`, and `py` for directional padding control.

<PreviewBox title="padding-directions.tsx" code={paddingDirectionsSnippet}>
  <Flex gap="2" align="center">
    <Box p="2" className="border border-[rgb(var(--border-1))]">
      p = 2
    </Box>
    <Box pt="2" className="border border-[rgb(var(--border-1))]">
      pt = 2
    </Box>
    <Box pb="2" className="border border-[rgb(var(--border-1))]">
      pb = 2
    </Box>
    <Box pl="2" className="border border-[rgb(var(--border-1))]">
      pl = 2
    </Box>
    <Box pr="2" className="border border-[rgb(var(--border-1))]">
      pr = 2
    </Box>
    <Box px="2" className="border border-[rgb(var(--border-1))]">
      px = 2
    </Box>
    <Box py="2" className="border border-[rgb(var(--border-1))]">
      py = 2
    </Box>
  </Flex>
</PreviewBox>

### Custom Padding

Padding also accepts custom CSS values.

<PreviewBox title="custom-padding.tsx" code={customPaddingSnippet}>
  <Box className="border border-[rgb(var(--border-1))] w-fit" p="17px">
    p = 17px
  </Box>
</PreviewBox>
