# Spacing Properties

Controls padding, margin, and spacing scale for elements. Supports scaled values, custom values, directional control, negative margins, and spacing inheritance.

## Padding

### Padding Scale

The `p` prop accepts scale values `0`–`8` for consistent spacing.

<PreviewBox title="padding-scale.tsx" code={paddingScaleSnippet}>
  <Flex gap="2" align="center">
    {paddingValues.map((num) => (
      <Box
        key={num}
        className="border border-[rgb(var(--border-1))] h-fit"
        p={`${num}`}
      >
        {num}
      </Box>
    ))}
  </Flex>
</PreviewBox>

### Padding Directions

Use `p`, `pt`, `pb`, `pl`, `pr`, `px`, and `py` for directional padding control.

<PreviewBox title="padding-directions.tsx" code={paddingDirectionsSnippet}>
  <Flex gap="2" align="center">
    <Box p="2" className="border border-[rgb(var(--border-1))]">
      p = 2
    </Box>
    <Box pt="2" className="border border-[rgb(var(--border-1))]">
      pt = 2
    </Box>
    <Box pb="2" className="border border-[rgb(var(--border-1))]">
      pb = 2
    </Box>
    <Box pl="2" className="border border-[rgb(var(--border-1))]">
      pl = 2
    </Box>
    <Box pr="2" className="border border-[rgb(var(--border-1))]">
      pr = 2
    </Box>
    <Box px="2" className="border border-[rgb(var(--border-1))]">
      px = 2
    </Box>
    <Box py="2" className="border border-[rgb(var(--border-1))]">
      py = 2
    </Box>
  </Flex>
</PreviewBox>

## Spacing

### Spacing Property

The `spacing` prop sets the overall spacing scale for an element: `xs`, `sm`, `md`, `lg`, or `xl`.

<PreviewBox title="spacing-property.tsx" code={spacingPropertySnippet}>
  <Flex gap="2" align="center">
    {spacingValues.map((size) => (
      <Box
        p="4"
        key={size}
        className="border border-[rgb(var(--border-1))]"
        spacing={size}
      >
        {size.toUpperCase()}
      </Box>
    ))}
  </Flex>
</PreviewBox>

### Spacing Inheritance

Child elements inherit the spacing scale from their parent.

<PreviewBox title="spacing-inheritance.tsx" code={spacingInheritanceSnippet}>
  <Box
    spacing="xl"
    className="border border-[rgb(var(--border-1))]"
    p="2"
    m="2"
  >
    Parent Box (XL Spacing, p=2, m=2)
    <Box className="border border-[rgb(var(--border-1))]" p="2" m="2">
      Child 1 (Default Spacing, p=2, m=2)
    </Box>
  </Box>
</PreviewBox>
